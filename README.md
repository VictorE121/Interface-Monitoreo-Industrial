# Industrial Monitoring Dashboard

Dashboard de monitoreo industrial que simula el estado de maquinaria en tiempo real, datos de producción y alertas del sistema. Diseñado para demostrar conceptos básicos de automatización industrial, monitoreo de equipos y visualización tipo SCADA.

---

## 🚀 Funcionalidades

- Estado de máquina en tiempo real (RUNNING / IDLE / ERROR)
- Datos de producción simulados con actualización en vivo
- Sistema básico de alertas por condiciones críticas
- Visualización de datos interactiva
- Interfaz estilo SCADA (modo oscuro)

---

## 🛠️ Tecnologías

- React (Vite)
- Recharts
- JavaScript (ES6)

---

## ⚙️ Instalación

```bash
npm install
npm run dev
```

---

## 📊 ¿Cómo funciona?

- Los datos se simulan usando actualizaciones periódicas (`setInterval`)
- El estado de la máquina cambia dinámicamente
- Se generan alertas cuando los valores superan ciertos límites
- Las gráficas se actualizan en tiempo real

---

## 🧠 Propósito

Este proyecto simula un sistema básico de monitoreo industrial y demuestra:

- Seguimiento del estado de equipos  
- Conceptos básicos de SCADA / HMI  
- Visualización de datos en tiempo real  
- Entendimiento de entornos de producción
